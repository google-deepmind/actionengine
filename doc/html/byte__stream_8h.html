byte_stream.h File Reference

Detailed Description

A wrapper to make an EvergreenStream from send/recv-like functions.

This file contains the definition of the EvergreenByteStream class, which is used to create an EvergreenStream from raw byte streams. It provides methods to send and receive messages, as well as to send and receive raw bytes.
